Это первый проект в рамках освоения новой профессии Frontend-разработчик.
Данный pet-проект представляет из себя одностраничный сайт-визитку, содержащий базовую информацию о разработчике сайта и его родном брате. Проект написан без использования фреймворков, только HTML, CSS и ванильный JavaScript. Сайт адаптирован под все устройства. Стилизация правильно работает в таких браузерах как Chrome, Opera, Firefox по версиям, соответствующим 2022 году.
